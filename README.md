# cpp-snippets

Collection of C++ examples and exercises organized by topics. Each folder
contains a `.cpp` version and, when applicable, a subdirectory with the same
exercise name.

## Structure

- `exercises/`: exercises organized in sequence (`01-10/`, `11-20/`, `21-30/`, `31-40/`).
- `challenges/`: LeetCode-style challenges.
- `snippets/`: small standalone examples.
- `study/`: study notes/experiments.
- `artifacts/`: local build/analysis outputs (ignored by git).

## Install tools (Linux)

Below are common ways to install a C++ compiler on Linux.

### Debian/Ubuntu

```bash
sudo apt update
sudo apt install build-essential
```

### Fedora

```bash
sudo dnf install gcc-c++
```

### Arch

```bash
sudo pacman -S base-devel
```

## Build and run

Example using GCC (g++):

```bash
g++ -std=c++17 exercises/21-30/26-rock-paper-scissors.cpp -o artifacts/bin/rps
./artifacts/bin/rps
```

Replace the `.cpp` path with the exercise you want.

If you want to compile with warnings enabled:

```bash
g++ -std=c++17 -Wall -Wextra -Wpedantic exercises/21-30/26-rock-paper-scissors.cpp -o artifacts/bin/rps
```

## Requirements

- C++ compiler (e.g., `g++`).

## Notes

- `tempCodeRunnerFile.cpp` is generated by some IDEs and can be ignored.
- Info about the code: ```mkdir -p artifacts/build_info artifacts/bin && g++ -O2 -g snippets/pointer-reinterpret-cast.cpp -o artifacts/bin/app && objdump -dS artifacts/bin/app > artifacts/build_info/disasm_with_source.txt && readelf -SW artifacts/bin/app > artifacts/build_info/sections.txt && nm -n artifacts/bin/app > artifacts/build_info/symbols.txt
```
- Info with warning: ```mkdir -p artifacts/build_info artifacts/bin && g++ -O2 -g snippets/pointer-reinterpret-cast.cpp -o artifacts/bin/app 2> artifacts/build_info/gpp_err.txt && objdump -dS artifacts/bin/app > artifacts/build_info/disasm_with_source.txt 2> artifacts/build_info/objdump_err.txt && readelf -SW artifacts/bin/app > artifacts/build_info/sections.txt 2> artifacts/build_info/readelf_err.txt && nm -n artifacts/bin/app > artifacts/build_info/symbols.txt 2> artifacts/build_info/nm_err.txt
```
